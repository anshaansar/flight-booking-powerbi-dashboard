# Flight Booking & Travel Data Analysis Dashboard

# Project Overview
This project is an interactive Power BI dashboard developed to analyze flight booking and travel data. It provides valuable insights into airline performance, ticket prices, flight distribution, travel classes, and booking patterns.

# Dashboard Features
- Total Flights
- Average Ticket Price
- Maximum Ticket Price
- Average Flight Duration
- Average Ticket Price by Airline
- Flights by Source City
- Flights by Destination City
- Flight Class Distribution
- Stops Distribution
- Interactive Filters (Airline, Source City, Destination City, Class)

# Tools Used
- Microsoft Power BI
- Microsoft Excel
- Data Visualization
- Data Analysis

# Dataset
Flight Booking & Travel Dataset

# Key Insights
- Compare average ticket prices across airlines.
- Analyze popular source and destination cities.
- Understand Economy vs Business class distribution.
- Analyze stop patterns and flight duration.
